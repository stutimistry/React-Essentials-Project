## React Essentials Project

A beginner-friendly React project built to practice and understand the core concepts of React.js.
This project focuses on reusable components, props, state management, event handling, and dynamic UI rendering.

## Features
Reusable React components
Dynamic content rendering
Interactive UI
Event handling
State management using useState
Responsive design
Clean component structure

## Tech Stack
React.js
JavaScript (ES6)
CSS3
Vite
